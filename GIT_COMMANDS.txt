  GITHUB PAGES DEPLOYMENT - COPY & PASTE THESE COMMANDS
═══════════════════════════════════════════════════════════════════

STEP 1: Stage all changes
────────────────────────────────────────────────────────────────────
git add .


STEP 2: Commit changes
────────────────────────────────────────────────────────────────────
git commit -m "Configure GitHub Pages deployment"


STEP 3: Push to GitHub
────────────────────────────────────────────────────────────────────
git push origin main


═══════════════════════════════════════════════════════════════════
  AFTER PUSHING - ENABLE GITHUB PAGES (FIRST TIME ONLY)
═══════════════════════════════════════════════════════════════════

1. Go to: https://github.com/<your-username>/cdtrc102
2. Click: Settings (top right)
3. Click: Pages (left sidebar)
4. Under "Source": Select "GitHub Actions"
5. Click: Save

═══════════════════════════════════════════════════════════════════
  MONITOR DEPLOYMENT
═══════════════════════════════════════════════════════════════════

1. Go to: https://github.com/<your-username>/cdtrc102/actions
2. Watch the "Deploy to GitHub Pages" workflow
3. Wait for green checkmark (2-3 minutes)
