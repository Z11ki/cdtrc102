# CDTRC Website

## Child Development Training & Research Center

A modern, professional website for CDTRC - a Non-Governmental Organization focused on training professionals who work with children and conducting research in child development.

## 🎨 Design Features

- **Primary Theme Color**: Sea Blue (#0891b2)
- **Accent Color**: Yellow (#fbbf24)
- **Main Background**: White
- **Clean, trustworthy, child-focused but professional design**
- **Soft rounded UI components**
- **Subtle glowing animations behind buttons**
- **Smooth fade-in and slide-in animations when scrolling**
- **Professional typography with modern sans-serif fonts**
- **Fully responsive design (mobile + tablet + desktop optimized)**

## 📄 Pages

### 1. Home Page
- Hero section with organization logo
- Strong headline: "Empowering Those Who Shape the Future of Children"
- Mission statement
- Call-to-action buttons with glow animation
- Impact statistics with animated counters
- Service highlights

- Organization background story
- Vision and Mission statements

- Grid layout of staff members

- Description of training programs

- Clean grid gallery layout

## ⚙️ Features

- **Sticky navigation bar** - Stays visible while scrolling
- **Smooth scrolling** - Seamless page navigation
- **Footer** - Quick links and copyright information
- **SEO optimized structure** - Semantic HTML
- **Accessibility friendly** - ARIA labels and keyboard navigation
- **Animations** - Motion/Framer Motion for smooth transitions
- **Responsive design** - Works on all device sizes

## 🚀 Technology Stack

- **React** - UI framework
- **React Router** - Navigation
- **Tailwind CSS** - Styling
- **Framer Motion** - Animations
- **Lucide React** - Icons
- **TypeScript** - Type safety

## 🎯 Design Philosophy

The website design reflects:
- **Trustworthiness** - Professional color scheme and layout
- **Child-focused but not childish** - Balanced design approach
- **NGO-standard quality** - Clean and credible presentation
- **Modern and minimal** - Contemporary design patterns
- **Sea blue gradients** - Subtle use throughout
- **Yellow accents** - For highlights and CTAs only

## 📱 Responsive Breakpoints

- Mobile: < 768px
- Tablet: 768px - 1024px
- Desktop: > 1024px
